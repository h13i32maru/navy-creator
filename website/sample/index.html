Getting GitHub token

サンプルアプリのGitHubClientを使用するためにはGitHub APIへアクセスするためのトークンを取得する必要があります。 https://github.com/settings/applications#personal-access-tokensにアクセスして 下記の手順でトークンを取得してください。

取得したトークンはコピー&ペーストでGitHubClientに貼り付けるため、トークンの取得はお使いのモバイル端末で行うことをおすすめします。

1. 「Generate new token」をクリックします

2. 「public_repo」を選択してトークンを生成します

3. トークンをコピーしてGitHubClientに貼り付けます
